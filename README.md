# aozyildirim/homebrew-tap

Homebrew tap for the [AGENA](https://agena.dev) platform CLI.

## Install

```bash
brew tap aozyildirim/tap
brew install agena
```

or as a one-liner:

```bash
brew install aozyildirim/tap/agena
```

## Formulas

| Formula | Description |
|---|---|
| `agena` | Official CLI — enroll your machine as a Runtime, run the CLI bridge, manage agents. |

## Updating

This tap is kept in sync by [GoReleaser](https://goreleaser.com/) from
the [main Agena repo](https://github.com/aozyildirim/Agena) on every
tagged release. `Formula/agena.rb` is rewritten automatically; manual
edits may be overwritten on the next release.
